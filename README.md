### 🏗️ Featured Project

#### 🤖 智能客服 Multi-Agent 协同系统
> 面向电商会员场景的 LLM 多 Agent 平台，日均处理 50万+ 请求

- **架构**：ReAct + Plan-and-Execute，Router Agent 动态调度订单 / 物流 / 退款专项 Agent
- **RAG 链路**：文档切片 → Embedding → Milvus 向量库 → 重排序，知识命中率 62% → **88%**，幻觉率下降 **40%**
- **Tool Calling**：封装 10+ 内部系统为标准化 Tool，平均问题解决轮次 5.2 → **2.1**
- **可观测**：LangSmith + 自研 Trace 平台，全链路追踪 Token / 工具调用 / 耗时
- **Human-in-the-loop**：低置信度决策自动转人工 + 反馈记录，构建数据飞轮，模型迭代周期 2周 → **3天**
